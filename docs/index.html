Splits

Interval training management for running coaches — automated timing, real-time rest countdowns, and performance reporting.

Download for macOS Download for Windows All releases

How It Works

Splits is built for running coaches who use interval training — repeated high-effort runs separated by structured rest periods. Before a session, the coach configures the interval distance and rest duration. During the workout:

  1. A runner begins their interval by tapping an NFC reader to register the start.
  2. When they cross the finish, their RFID tag is automatically detected, stopping the clock and logging their split time.
  3. The runner immediately sees a countdown showing how much rest time remains before their next interval.
  4. When rest time is up, the app alerts the runner that it's time to go again.

After the session, the coach can generate an HTML performance report for each athlete and email it directly from the app using their Google account.

Features

Automated Interval Timing

Interval start is registered with an NFC tap; finish is detected automatically via each runner's RFID tag — no manual stopwatch or human error.

Real-Time Rest Countdown

As soon as a runner finishes, they see exactly how much rest time they have left and receive an alert when it's time to start their next interval.

Session Configuration

Coaches set the interval distance and rest duration before each workout. The app handles all the timing from there.

Roster Management

Import your athlete roster and associate each runner with their RFID tag so every split is logged to the right person automatically.

HTML Performance Reports

Generate rich HTML reports after each session, showing split times and performance summaries for every athlete.

Email Reports via Gmail

Sign in with your Google account to send reports directly to athletes from within the app — no copy-pasting required.

System Requirements

  • macOS macOS 10.14 or later. NFC timing requires pcsc-lite (brew install pcsc-lite).
  • Windows Windows 10 or later. Installer includes a desktop shortcut and uninstaller.
